# Legacy claude.md Archive

> 이 폴더는 마이그레이션된 레거시 claude.md 파일 보관용입니다.

## 정책

- **운영 SSOT 아님**: 이 폴더의 파일은 참고용이며, 운영 규칙으로 사용하지 않습니다.
- **현재 SSOT**: [docs/03_ops/AGENTS.md](../../03_ops/AGENTS.md)
- **마이그레이션**: `./scripts/install.sh --migrate-claude-md` 명령으로 자동 이동됨

## 파일명 규칙

원래 경로가 파일명으로 인코딩됩니다:
- `apps/web/CLAUDE.md` → `apps__web__CLAUDE.md`
- `src/auth/claude.md` → `src__auth__claude.md`

## 참고
